# MediSync Frontend Images

## Required Images

Place the following images in this folder:

### Background Images

- `bg-welcome.png` - Welcome screen background
- `bg-vitals.png` - Vitals monitoring screen background
- `bg-symptoms.png` - Symptoms selection screen background

### Medicine Images

- `pill-neozep.png` - Neozep medicine icon
- `pill-biogesic.png` - Biogesic medicine icon
- `pill-buscopan.png` - Buscopan medicine icon
- `pill-cetirizine.png` - Cetirizine medicine icon

### Logo

- `logo.png` - MediSync logo

## Image Requirements

- **Format**: PNG with transparency preferred
- **Size**: Optimize for web (under 500KB each)
- **Dimensions**:
  - Backgrounds: 1920x1080px or higher
  - Pills/Icons: 256x256px or 512x512px
  - Logo: 200x200px

## Placeholder Mode

If images are not available, the UI will use:

- Color-coded cards instead of images
- Text labels for medicines
- Simple emoji icons as fallbacks

## Adding Images

1. Place your image files in this folder
2. Ensure filenames match exactly as listed above
3. Update image imports in component files if needed
